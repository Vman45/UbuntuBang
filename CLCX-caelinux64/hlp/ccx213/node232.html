*DAMPING

Keyword type: model definition

This card is used to define Rayleigh damping for implicit dynamic calculations (*DYNAMIC). There are two required parameters: ALPHA and BETA.

Rayleigh damping is applied in a global way, i.e. the damping matrix is taken to be a linear combination of the stiffness matrix and the mass matrix :

(456)

Right now, only one *DAMPING card can be used in the input deck. It applies to the whole model.


First line:

  • *DAMPING
  • Enter ALPHA and BETA and their values. 

Example:

*DAMPING,ALPHA=5000.,BETA=2.e-3

indicates that a damping matrix is created by multiplying the mass matrix with and adding it to the stiffness matrix multiplied by
