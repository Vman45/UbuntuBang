wsize 

   'wsize' [RETURN]|'f']|[<xp> <yp>]
Size of the window were xp and yp are the size in pixel. Without argument the size is the initial size and with the argument 'f' (fullsize) the screen resolution is used. This command takes no effect during reading and execution of a batch file. It will be executed in the glut event loop (the glut library [2] for window management and event handling).
